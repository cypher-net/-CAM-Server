=|CAM|= Club Amigos del Medalla

Este Readme intenta dar instrucciones especificas para la instalacion
y ejecucion tanto del juego como de los servidores.


-----
MAPAS
-----
Todo mapa nuevo que quiera ser jugado en el servidor (archivos .pk3) se tendra que colocar
en la carpeta /main y NO en otro lugar.

--------------
SERVIDORES CAM
--------------
Configuracion por defecto ejecutando los bash scripts (.bat)
Existen actualmente 2 bash para iniciar dos servidores en un mismo
host o ip

Ejecutar Server La Arena
Descripcion:
Al ejecutar el archivo bat, configura un puerto de escucha y luego ejecuta el
archivo de configuracion cam-server-arena.cfg y este ejecuta el cam-rconpassword.cfg

Ejecutable: moh_breakthrough_CAM_Arena.bat
Puerto por configuracion: 12203
Arch. Configuracion: /maintt/cam/cam-server-arena.cfg
Arch. rconpassword: /maintt/cam/cam-rconpassword.cfg
